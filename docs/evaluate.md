# モデルの評価方法
---

- src/train.pyでは、最後に評価ループが実行されるので、自動的に客観評価指標が算出されます。
- 結果は実験ディレクトリのresultsフォルダに保存されます。
- 評価セットを変更して再評価したい場合などは次のコマンドで実行することができます。
```bash
uv run python src/evaluate.py --exp=[実験ディレクトリのパス] --metrics=[評価セット]
```
- --metricsには、conf/metrics以下の設定ファイルの名前を指定してください。
- 評価セットにnoneを指定した場合は、RTF・消費メモリ量・パラメータ数のみが計算されます。
- --deviceにはcpu又はgpuを設定できます。
